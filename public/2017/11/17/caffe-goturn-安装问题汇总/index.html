caffe - goturn 安装问题汇总

1.caffe目录未找到

修改 ～/GOTURN/cmake/Modules/FindCaffe.cmake文件第5行，将caffe绝对路径加上即可。

https://github.com/davheld/GOTURN/issues/4

2. 未找到 caffe.pb.h

$$/home/embedded/caffe/include/caffe/blob.hpp:9:34: fatal error: caffe/proto/caffe.pb.h: No such file or directory$$

解决方法：

**用protoc从caffe/src/caffe/proto/caffe.proto生成`caffe.pb.h和caffe.pb.cc**`

http://blog.csdn.net/xmzwlw/article/details/48270225

3.boost regex 问题

4.tx2 安装 opencv失败

解决办法，缺啥装啥，

5.waring 　it doesnt matter

6.vot

解决办法

  • 首先下载trax,

    git clone https://github.com/votchallenge/trax.git

  • 然后编译trax

  • 修改CMakeLists.txt of GOTURN around line 84:把trax的实际路径写上

https://github.com/davheld/GOTURN/issues/1
