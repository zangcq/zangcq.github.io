* 宏定义 描述文件 zzz.md

```
 
      // define fpga_instruction
      #ifndef DEF_ZZZ
      #error "Must define ZZZ first"
      #endif
      // DEF_ZZZ(FIELD,BITSIZE)
      DEF_ZZZ(one,1)
      DEF_ZZZ(two, 1)
      DEF_ZZZ(three, 1)
      DEF_ZZZ(four, 1)
```

  * 定义结构体 头文件 zzz.h

```
 
      struct ZZZ {
      #define ZZZ(field, bitsize) \
         unsigned int field : bitsize;
      #include "zzz.md"
      #undef DEF_ZZZ
      }__attribute__((packed));
```

  * ToString 方法 .cc 文件 zzz.cc

